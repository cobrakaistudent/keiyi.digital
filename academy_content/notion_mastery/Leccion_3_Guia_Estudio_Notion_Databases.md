# Guía de Estudio: Notion Mastery - El Motor de Proyectos (Día 3)
**Taller:** Notion Mastery | **Día 3**
**Objetivo:** Dominar las Bases de Datos para gestionar Proyectos y Docs. Aprenderás por qué Notion es la herramienta preferida por las startups para centralizar el trabajo.

---

## 1. Bases de Datos: Más que simples Tablas
En Notion, una base de datos es una colección de páginas con **Propiedades** (metadatos).
*   **Wikis Dinámicas:** Puedes convertir una tabla de documentos en una Wiki donde cada entrada tiene un responsable y una fecha de última revisión.
*   **Vistas:** Una misma base de datos puede ser una Tabla, una Galería (visual) o una Lista (minimalista).

## 2. Propiedades Inteligentes
Las propiedades son el cerebro de Notion. Configuraremos las esenciales para proyectos:
*   **Status:** Para saber si el proyecto está "Iniciando", "En Pausa" o "Finalizado".
*   **Priority:** Usando emojis para identificar lo urgente.
*   **Due Date:** La fecha límite que se sincronizará con tu calendario (lo veremos mañana).

## 3. Notion AI: El Asistente de Proyectos
Aprovecharemos las capacidades de **Notion AI Autofill**:
*   Notion puede resumir automáticamente el contenido de una página de proyecto dentro de una propiedad de la base de datos.
*   Puede extraer "Action Items" (tareas pendientes) de una minuta de reunión sin que tú las escribas.

---

## 4. Laboratorio del Día 3: Tu Motor de Proyectos
**Misión:** Crear una base de datos de Proyectos profesional.
1.  **Crea una Tabla de Proyectos:** Añade propiedades de *Status*, *Date* y *Priority*.
2.  **Vista Galería:** Crea una vista alternativa donde cada proyecto se vea como una tarjeta visual. Usa el "Page Cover" como previsualización.
3.  **Primer Doc:** Crea un proyecto, ábrelo como página y usa el comando `/ai` para pedirle que genere un "Esquema de Proyecto" (Project Outline) para ti.

---
**Resultado:** Has pasado de simples notas a un sistema de gestión dinámico. Ahora tus proyectos tienen estructura, fechas y un asistente de IA trabajando contigo.

# Guía de Estudio: Notion Mastery - Relaciones y Formulas 2.0 (Día 5)
**Taller:** Notion Mastery | **Día 5**
**Objetivo:** Conectar tus bases de datos para crear un ecosistema inteligente. Usaremos **Relaciones, Rollups y las nuevas Formulas 2.0** para automatizar el cálculo de progreso.

---

## 1. Relaciones: El Vínculo Maestro
Las bases de datos no deben ser islas. Una **Relación** te permite conectar una Tarea con un Proyecto, o un Cliente con una Factura.
*   **Two-way Relation:** Al conectar Tareas con Proyectos, puedes ver todas las tareas desde el proyecto y el nombre del proyecto desde cada tarea.

## 2. Rollups: Extracción de Inteligencia
Un Rollup te permite ver datos de una base de datos relacionada sin salir de la actual.
*   **Ejemplo:** En tu base de Proyectos, usa un Rollup para contar cuántas Tareas relacionadas están marcadas como "Hecho".

## 3. Formulas 2.0: La Magia Visual
Con las nuevas Formulas 2.0 de Notion, podemos crear barras de progreso elegantes.
*   **Barra de Progreso Automática:** Usaremos una fórmula simple que tome el porcentaje de tareas terminadas y dibuje una barra visual ( ▓▓▓░░░ ) directamente en tu tabla de proyectos.
*   **Días Restantes:** Crearemos una fórmula que te diga cuántos días quedan para el deadline de forma amigable (ej. "¡Quedan 3 días!").

---

## 4. Laboratorio del Día 5: Conectando el Cerebro
**Misión:** Crear una conexión inteligente entre Proyectos y Tareas.
1.  **Crea la Relación:** Conecta tu base de Tareas con la de Proyectos.
2.  **Configura el Rollup:** En Proyectos, añade un Rollup que muestre el "% de Tareas Terminadas".
3.  **Primer Formula 2.0:** Copia y pega la fórmula de barra de progreso que te daremos en el material de apoyo para visualizar el avance de tus proyectos.

---
**Resultado:** Tu Notion ya no es estático. Ahora calcula, resume y te muestra visualmente qué tan cerca estás de tus metas.
